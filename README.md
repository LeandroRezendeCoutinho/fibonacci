# Fibonacci algorithms

## Languages
* C
* Crystal
* Elixir
* Javascript
* Python
* Ruby
* Rust
* Typescript


### First check if you have the language installed

#### Running C
```
gcc fib.c  -o fibc
./fibc
```

#### Running Crystal
```
crystal run fib.cr
```

#### Running Crystal optimized
```
crystal build --release fib.cr -o fibcr
./fibcr
```

#### Running Elixir
```
elixir fib.ex
```

#### Running Javascript
```
node fib.js
```

#### Running python
```
python fib.py
```

#### Running Ruby
```
ruby fib.rb
```

#### Running Ruby with JIT
```
ruby --jit fib.rb
```

#### Running Rust
```
rustc fib.rs -o fibrs
./fibrs
```

#### Running Typescript
```
deno fib.ts
```
